    • Each image is graded with a KL score (although you are supposed to be familiar with this (i.e. OA related material on the webpage), if you are not familiar read below for a short introduction.)
    KL grading (click to expand)

     #from https://radiopaedia.org/articles/kellgren-and-lawrence-system-for-classification-of-osteoarthritis-of-knee
    The Kellgren and Lawrence (KL) system is a common method of classifying the severity of knee osteoarthritis (OA) using five grades 1. This classification was proposed by Kellgren et al. in 1957 2 and later accepted by WHO in 1961.

    In applying the system, the prevalence of chronic knee pain in a postal survey of 2000 Swedes aged 35-54 was 15%1. Of these respondents, 1% showed radiographic knee OA, based on bilateral weight bearing plain films1.

    Classification

    grade 0: no radiographic features of OA are present
    grade 1: doubtful joint space narrowing (JSN) and possible osteophytic lipping
    grade 2: definite osteophytes and possible JSN on anteroposterior weight-bearing radiograph
    grade 3: multiple osteophytes, definite JSN, sclerosis, possible bony deformity
    grade 4: large osteophytes, marked JSN, severe sclerosis and definite bony deformity

    The grades are stored in this following file baseline.csv. It is a table that shows different measures and outcomes for each id.

    Details about image assessment can be found here.

    Use pandas to read that (.csv) file and check the following colums for KL grading:

      'ID' : subject ID
  'V00XRKL' : grade
  'SIDE' : side (1: right, 2: left)

    There are 5 different grades: 0,1,2,3,4.

    Use following labels for multi-class classification

    • Class 1: Grade0 ,
    • Class 2: Grade1

    For binary classification, you can combine grades<2 and grades>1:

    • Class 1: Grade0, Grade1
    • Class 2: Grade2, Grade3, Grade4

    You can use .sas7bdat file to obtain KL grades as well. Read from here: https://oai.epi-ucsf.org/datarelease/SASDocs/kXR_SQ_BU_descrip.pdf

    Use the following kxr_sq_bu00.sas7bdat

    You can use the following code:

    def read_kl_grades(fname='./kXR_SQ_BU00_SAS/kxr_sq_bu00.sas7bdat'):

fp = pandas.read_sas( fname, format = 'sas7bdat', encoding='iso-8859-1')

#get column names and find entries that end with 'KL'
lfp = list(fp)
matches = [x for x in lfp if 'KL'== x[-2:]]

fp['ID'] = fp['ID'].astype('int')
fp['SIDE'] = fp['SIDE'].astype('int')

#add 'ID' index
fp_kl = fp[ ['ID'] + ['SIDE'] + matches]
fp_kl.drop_duplicates(subset=['ID', 'SIDE'], inplace=True)
#change index
kl_table = fp_kl.set_index('ID')


return kl_table
