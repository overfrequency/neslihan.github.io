    Environment

    Setting up an environment#

    The operating system#

    Install Ubuntu:
    https://tutorials.ubuntu.com/tutorial/tutorial-install-ubuntu-desktop#0

    Learn and Create a Virtual Environment#

    Install Anaconda & Python (3)
    https://conda.io/docs/user-guide/getting-started.html

    Install Python packages#

    Use virtual environment and conda to install scientific computing libraries.

    conda install numpy
conda install scipy
conda install matplotlib
    • Learn numpy
    • http://www.numpy.org/
    • Learn scipy
    • https://www.scipy.org

    Install Machine and Deep Learning packages.#

    Install Scikit-learn

    https://scikit-learn.org/stable/

    Select one of the deep learning library: - Pytorch
    - Tensorflow
    - Keras (Tensorflow)
    - Fast.ai (https://www.fast.ai/)

    Python IDE#

    I recommend PyCharm
